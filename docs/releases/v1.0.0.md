# Vector Engine v1.0.0 (Draft)

## Release Positioning

v1.0.0 is the first stability-oriented release with performance-first evidence. It emphasizes reproducible benchmark programs, explicit API stability gates, and publishable artifact protocols.

## Highlights

- Performance benchmark matrix:
  - scripted matrix runner (`scripts/benchmark_matrix.py`)
  - per-config benchmark artifacts
  - aggregate summary with backend-level stats
- Reproducibility evidence:
  - exact-equivalence reports (`artifacts/faiss_equivalence/`)
  - stability run summaries (`artifacts/testing_runs/`)
  - matrix summary artifacts (`artifacts/benchmark_matrix/matrix_summary.json`)
- API stability gate:
  - documented v1.0 public contract in `docs/api.md`
  - compatibility regression tests in `tests/test_api_stability.py`

## Caveats

- Benchmark claims are configuration- and hardware-dependent.
- Real private-corpus embeddings and sensitive ID mappings remain out of repo.
- Publish summary artifacts, not confidential input payloads.

## Migration Notes (from v0.3.x)

- No intentional breaking changes to core public APIs.
- v1.0 docs formalize API stability contract and reproducibility protocol.
- Existing v0.3 workflows remain valid; additional benchmark matrix tooling is now available.
- Reference artifact: `artifacts/benchmark_matrix/publishable_results.v1.json`.

## Reproduce Release Numbers

```bash
python scripts/rag_baseline.py --output-dir artifacts --k 3
python scripts/rag_real_corpus_eval.py --embeddings ... --query-embeddings ... --ids ... --ground-truth ... --output artifacts/real_corpus_runs/run_1.json --backend bruteforce --k 10 --ks 1,5,10 --loops 5
python scripts/stability_runs.py --embeddings ... --query-embeddings ... --ids ... --ground-truth ... --backend bruteforce --run-count 200 --output-dir artifacts/testing_runs
python benchmarks/compare_bruteforce_vs_faiss.py --mode exact --min-flat-overlap 0.99 --output artifacts/faiss_equivalence/run_1.json
python scripts/benchmark_matrix.py --mode exact --warmup 2 --loops 8 --seed 7 --min-flat-overlap 0.99 --output-dir artifacts/benchmark_matrix
python scripts/publishable_results.py --matrix-summary artifacts/benchmark_matrix/matrix_summary.json --stability-summary artifacts/testing_runs/stability_summary_bruteforce_200.json --output artifacts/benchmark_matrix/publishable_results.v1.json
```
